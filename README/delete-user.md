### 権限
必要

### エンドポイント
```
DELETE https://goal-management.com/api/user/delete
```


### パラメータ
| 場所   | 名称             | 内容                 |
|------|----------------|--------------------|
| ヘッダー | `X-XSRF-TOKEN` | ログイン時に取得したCSRFトークン |

### 応答
| 名称        | 内容        |
|-----------|-----------|
| `message` | 成功時のメッセージ |

### 例
```js
{
    message:"ユーザーデータを全て削除しました。"
}
```
