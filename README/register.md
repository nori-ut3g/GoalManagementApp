### エンドポイント
```
POST https://goal-management.com/api/register
```

### パラメータ
| 場所  | 名称         | 内容             |
|-----|------------|----------------|
| 本文  | `name`     | ユーザーの名前　Max:20 |
| 本文  | `email`    | メールアドレス　Unique |
| 本文  | `password` | パスワード　Min:8    |

### 応答
| 名称        | 内容            |
|-----------|---------------|
| `message` | 成功時のメッセージ |

### 例
```js
{
    message:"ユーザ登録しました"
}
```
